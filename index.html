Automation Spotlight

AI & Automation Corner 🤖

What the team is automating, plus a personal-level tip you can use today.

🤖
Jira → Excel Automation
Python · Atlassian API · IBUNBEES

Built a Python script that pulls IBUNBEES tickets from Jira Cloud API and dumps them straight into Excel — replacing what used to be a 40-minute manual export. Uses pagination via nextPageToken on the new /search/jql endpoint.

💡 Personal Automation Tip of the MonthStop copy-pasting Outlook calendar to Excel. Use Power Automate's "When an event is added" trigger → "Add row to Excel" action. Setup takes 5 minutes. Saves hours every week.
